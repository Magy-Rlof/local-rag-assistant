# 示例岗位：初级 AI 应用开发工程师

## 岗位定位

该岗位面向 AI 应用落地，重点关注如何把大模型能力接入业务系统，而不是从事大模型底层训练或算法研究。

## 主要职责

- 调用大模型 API 完成业务问答或内容生成
- 参与 RAG 知识库问答系统开发
- 编写 Prompt 并优化模型输出格式
- 将 AI 能力接入后端服务或内部工具
- 整理项目文档、运行说明和局限性说明

## 能力要求

- 熟悉 Python 基础开发
- 理解 API Key 安全管理
- 理解 Prompt、结构化输出和 Tool Calling 基本概念
- 了解 RAG 的文档加载、切分、检索和上下文拼接流程
- 能够清楚说明项目的业务价值和技术边界

## 加分项

- 有本地文档问答 Demo
- 有可运行的 GitHub 项目
- 了解 LangChain、LlamaIndex 或 Dify
- 有 Java 后端或 Web 系统接入经验

## 不重点要求

该岗位不重点要求大模型训练、CUDA 优化、论文复现或传统 NLU 模型训练。
