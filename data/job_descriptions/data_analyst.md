# 示例岗位：数据分析师

> 说明：本文档为模拟示例数据，仅用于演示 RAG 检索流程，不代表实时招聘市场。

## 岗位定位

数据分析师通过数据处理、指标分析、报表和业务解释支持决策。在 AI 应用场景中，数据分析师可以利用大模型辅助生成分析摘要、解释指标变化和整理报告。

## 主要职责

- 清洗和整理业务数据
- 设计指标口径和分析报表
- 发现业务问题并提出解释
- 编写分析报告和汇报材料
- 使用 AI 工具辅助总结和问答

## 能力要求

- 熟悉 SQL 和表格处理
- 理解常见业务指标
- 能用 Python 做基础数据处理
- 能清楚表达分析结论和局限性
- 了解 AI 工具在报告生成和知识问答中的应用

## 与 AI 应用的结合点

- 使用 RAG 查询指标口径文档
- 让模型基于数据说明生成报告草稿
- 对分析结论进行结构化输出
- 构建业务知识库问答助手
