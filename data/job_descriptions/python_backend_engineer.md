# 示例岗位：Python 后端工程师

> 说明：本文档为模拟示例数据，仅用于演示 RAG 检索流程，不代表实时招聘市场。

## 岗位定位

Python 后端工程师负责开发业务接口、数据处理任务和后台服务，在 AI 应用项目中常承担模型 API 接入、任务编排和服务封装工作。

## 主要职责

- 开发 REST API、后台任务和数据处理服务
- 对接数据库、缓存和第三方接口
- 封装大模型 API 调用和错误处理逻辑
- 支持文档解析、任务队列和异步处理
- 编写接口文档和基础测试

## 能力要求

- 熟悉 Python 基础语法和常用 Web 框架
- 理解 HTTP、JSON、鉴权和异常处理
- 熟悉数据库基本操作
- 能阅读需求并拆分接口任务
- 对 AI API、RAG 或自动化脚本有基本了解

## 与 AI 应用的结合点

Python 后端可以负责模型调用服务、Embedding 生成、向量库读写、文档处理和 Agent 工具接口。
