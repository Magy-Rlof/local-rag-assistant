# 示例岗位：DevOps/运维工程师

> 说明：本文档为模拟示例数据，仅用于演示 RAG 检索流程，不代表实时招聘市场。

## 岗位定位

DevOps/运维工程师负责部署、监控、日志、故障处理和运行稳定性。在 AI 应用项目中，需要关注模型服务、向量库和后端接口的可用性。

## 主要职责

- 配置应用运行环境和部署流程
- 管理日志、监控和告警
- 处理服务异常和性能问题
- 维护环境变量和密钥配置
- 支持数据库、缓存和向量库运行

## 能力要求

- 熟悉 Linux、网络和基础脚本
- 理解服务部署、端口、进程和日志
- 了解 Docker、CI/CD 或云服务更佳
- 重视安全配置和访问权限
- 能排查 API 超时、鉴权失败和服务不可用问题

## 与 AI 应用的结合点

DevOps 可以保障模型 API 调用、向量库索引、后端接口和前端服务稳定运行。
