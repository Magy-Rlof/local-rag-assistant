# 示例岗位：Java 后端工程师

> 说明：本文档为模拟示例数据，仅用于演示 RAG 检索流程，不代表实时招聘市场。

## 岗位定位

Java 后端工程师负责业务系统后端开发、接口设计、数据处理和系统稳定性建设。在 AI 应用项目中，Java 后端也可能负责把大模型服务接入现有业务系统。

## 主要职责

- 开发和维护后端 API
- 设计数据库表结构和业务流程
- 对接第三方服务或内部系统
- 处理权限、日志、异常和性能问题
- 在 AI 应用场景中封装模型调用接口

## 能力要求

- 熟悉 Java、Spring Boot 和 REST API
- 理解数据库、缓存和消息队列基础
- 具备接口联调和问题排查能力
- 能读懂 AI 服务接口文档并完成接入
- 有基础安全意识，能保护密钥和敏感配置

## 与 AI 应用的结合点

- 把 RAG 或 Agent 能力封装成后端服务
- 将模型回答接入企业业务流程
- 管理用户权限、调用日志和费用统计
- 为前端或内部系统提供统一 AI API
