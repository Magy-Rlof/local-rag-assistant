# 示例岗位：AI 应用开发工程师

> 说明：本文档为模拟示例数据，仅用于演示 RAG 检索流程，不代表实时招聘市场。

## 岗位定位

AI 应用开发工程师关注把大模型能力接入业务系统，重点是 API 调用、RAG、工具调用、工作流编排和应用落地，不以大模型底层训练为主要职责。

## 主要职责

- 调用大模型 API 完成问答、摘要、分类或内容生成
- 开发 RAG 知识库问答功能
- 设计 Prompt 和结构化输出格式
- 将 AI 能力接入 Web 系统、内部工具或自动化流程
- 记录系统局限性和运行方式

## 能力要求

- 熟悉 Python 基础开发
- 理解 API Key 安全管理
- 理解 Prompt、结构化输出和 Tool Calling
- 了解文档加载、文本切分、Embedding 和向量检索
- 能说明项目业务价值、技术边界和改进方向

## 加分项

- 有本地文档问答或知识库问答 Demo
- 使用过 Qdrant、Milvus、Chroma 或 pgvector
- 了解 Dify、LangChain 或 LlamaIndex
- 有后端接口开发经验

## 不重点要求

该岗位通常不重点要求大模型训练、CUDA 优化、论文复现或传统 NLU 模型专项训练。
