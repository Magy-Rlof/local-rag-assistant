## 求职目标

目标岗位为 Java 后端工程师、行业软件开发工程师或 AI 应用开发工程师，希望把后端开发能力迁移到 AI 应用落地场景。

## 技能概览

- 熟悉 Java 基础、Spring Boot、REST API 和 MySQL
- 理解接口设计、参数校验、异常处理和日志记录
- 了解 Python 脚本和大模型 API 调用
- 理解 RAG 的文档加载、向量检索和上下文拼接流程
- 能与前端和业务方沟通需求
