# 示例简历：初级 AI 应用开发方向

> 说明：本文档为脱敏模拟简历，仅用于演示 RAG 检索流程，不代表真实个人经历。

## 求职目标

目标岗位为初级 AI 应用开发工程师、Python 后端工程师或行业软件开发工程师，希望参与大模型 API 接入、知识库问答和业务工具开发。

## 技能概览

- 熟悉 Python 基础语法、requests、FastAPI 基础接口开发
- 理解 API Key 环境变量管理和错误处理
- 了解 Prompt 设计、结构化输出、RAG 和 Tool Calling 基本概念
- 使用过 Qdrant 本地向量检索和 BAAI/bge-m3 Embedding
- 能用 React 构建基础 Web 工具页面

## 项目经历

Local RAG Assistant：实现资料上传、增量索引、Qdrant 检索、引用来源展示、多轮对话和公开/私有资料管理。

## 可优化方向

简历应突出完整链路：文档加载、切分、Embedding、向量库、检索、Prompt 拼接、模型回答和前端交互。
