# 示例学习笔记：API 调用与 Prompt

> 说明：本文档为模拟示例数据，仅用于演示 RAG 检索流程。

## API 调用

AI 应用开发的第一步通常是调用大模型 API。程序需要读取用户输入，组织请求参数，调用模型服务，并解析返回结果。

## API Key 安全

API Key 是访问模型服务的凭证，不应写死在代码中，也不应提交到 GitHub。学习 Demo 应通过环境变量读取 API Key。

## Prompt 设计

Prompt 用于描述任务、背景、约束和输出格式。清晰的 Prompt 能让模型更稳定地完成任务。

## 结构化输出

结构化输出通常使用 JSON，让模型结果更容易被程序解析。实际项目中需要处理模型没有返回合法 JSON 的情况。
