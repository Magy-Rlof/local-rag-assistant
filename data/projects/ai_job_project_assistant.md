# 示例项目：AI 求职与项目知识库助手

> 说明：本文档为模拟示例数据，仅用于演示 RAG 检索流程。

## 项目定位

AI 求职与项目知识库助手是一个面向个人求职场景的 AI 应用原型。它用于管理项目说明、岗位 JD 和脱敏简历片段，并通过问答方式帮助用户整理求职材料。

## 核心功能

- 项目 README 问答
- 岗位 JD 技能提取
- 项目与岗位匹配分析
- 面试问题生成
- 简历项目描述建议

## 技术路线

项目使用 Python 完成后端原型，调用大模型 API 生成回答。RAG 部分从本地 Markdown 文档问答开始，逐步扩展到 Embedding、Qdrant 向量数据库和多文档检索。

## 使用场景

用户可以把脱敏后的项目说明和岗位 JD 放入知识库。系统根据用户问题检索相关资料，并生成回答、总结或面试准备建议。个人复盘资料应保留在本地私有目录，不作为默认知识库资料。

## 局限性

当前示例不包含真实简历、真实岗位数据或生产环境权限系统。回答质量依赖文档内容、Embedding 检索效果和大模型能力。

## 后续计划

后续可以增加文件上传、可视化界面、引用来源详情、JD 结构化分析和项目匹配评分。
